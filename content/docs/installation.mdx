### Install dependencies

Enter in the folder and install dependencies for your project:

```bash
cd my-saas-project
pnpm install
```

### Create a `.env` file

Copy/paste the `.env.example` in the `.env` file:

```md title=".env"
NEXT_PUBLIC_APP_URL=http://localhost:3000


```
